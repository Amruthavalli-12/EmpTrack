# 🚀 EmpTrack - Advanced Employee Management System

EmpTrack is a high-end, full-stack management ecosystem. It streamlines workplace operations with a futuristic UI and robust backend integration.

---

## 🛠️ Tech Stack Used
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** LocalStorage / JSON API
* **Icons:** FontAwesome 6.4

---

## ✨ Top Features
1. **Dual Portal:** Separate login flows for Admin and Employee.
2. **Dynamic CRUD:** Add, Edit, and Delete employee records in real-time.
3. **Security:** Secure password reset and ID verification.
4. **Modern UI:** Interactive glass-style cards with 3D animations.

---

## 📂 How to Setup
1. **Clone:** `git clone https://github.com/Amruthavalli-12/EmpTrack.git`
2. **Install:** `cd backend && npm install`
3. **Run:** Open `login.html` in your browser.
